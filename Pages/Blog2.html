Pro 3 SE

The Pro 3 is a 3 osc paraphonic synth from Sequential Circuits It is a paraphonic analog synth with two analog ocsillators and one digital wavetable. It has one filter per voice but there are three models to choose from. This synth has many great features which I dont have time to talk about here. but it sounds great and I use it all the time.
